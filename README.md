# Web Security Playground

A controlled Django application demonstrating common web vulnerabilities
and their secure implementations.

This project focuses on **understanding vulnerabilities and fixes**, not exploitation.

---

## What This Project Demonstrates

- Common web vulnerabilities
- Secure coding practices
- Defensive thinking
- OWASP Top 10 concepts

---

## Security Topics Covered

- SQL Injection
- Cross-Site Scripting (XSS)
- Cross-Site Request Forgery (CSRF)
- Broken access control
- Secure configuration

---

## Core Features

- Vulnerable implementations (controlled)
- Secure implementations side-by-side
- Clear explanations of attack vectors
- Demonstration of proper fixes
- Defensive coding patterns

---

## Why This Project Exists

Understanding vulnerabilities is meaningless without understanding how to fix them.
This playground focuses on **secure design and remediation**.

---

## Intended Audience

- AppSec Engineers
- Web Developers
- Security learners
- Code reviewers

---

## Disclaimer

All vulnerabilities are intentionally controlled and exist for educational purposes only.
